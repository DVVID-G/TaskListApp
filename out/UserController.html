Class: UserController

UserController()

Controller class for managing User resources. Extends the generic GlobalController to inherit CRUD operations, using the UserDAO as the data access layer.

Constructor

new UserController()

Create a new UserController instance. The constructor passes the UserDAO to the parent class so that all inherited methods (create, read, update, delete, getAll) operate on the User model.
